## 例题 1：1~5 号球同时取出 3 个，记 X 为最大编号，求 X 的分布律

X 的取值为 `3,4,5`

根据古典概型的概率计算得

```
P{X = 3} = C₃³ / C₅³ = 1/10
P{X = 4} = C₃² / C₅³ = 3/10
P{X = 5} = C₄² / C₅³ = 6/10
```

则 X 的分布律为

| X | 3 | 4 | 5 |
| :--: | :--: | :--: | :--: |
| P | 0.1 | 0.3 | 0.6 |

## 例题 4：某药有效率 0.95，现 10 人服用，求至少 8 人治愈的概率

```
X~B(10,0.95)

P{X = 8}=C₁₀⁸ × 0.95⁸ × 0.05²
P{X = 9}=C₁₀⁹ × 0.95⁹ × 0.05¹
P{X = 10}=C₁₀¹⁰ × 0.95¹⁰ × 0.05⁰

P = P{X = 8} + P{X = 9} + P{X = 10}
```

## 例题 5：X~B(2,p), Y~B(3,p), P{X ≥ 1} = 5/9，求 P{Y ≥ 1}

```
X 可取 0,1,2

由 P{X ≥ 1} = P{X = 1} + P{X = 2} = 5/9
得 C₂¹ × p¹ × (1-p)¹ + C₂² × p² × (1-p)⁰ = 5/9

2p(1-p) + p² = 5/9
2p - p² = 5/9
18p - 9p² = 5
9p² - 18p + 5 = 0
```

```
// 或
P{X = 0} = 4/9
C₂⁰ × p⁰ × (1-p)² = 4/9
```

```
p = 1/3 或 5/3，取 1/3
得 Y~B(3,1/3)

P{Y ≥ 1} = 1 - P{Y = 0} = 1 - C₃⁰ × (1/3)⁰ × (1-1/3)³ = 19/27
```

## 例题 6：X 服从 λ 的泊松分布，P{X = 1} = 2/3 P{X = 3}，求 λ

```
λ¹ / 1! × e^{-λ} = 2/3 × (λ³ / 3!) × e^{-λ}
λ = 2/3 × (λ³ / 6)
λ = 1/9 × λ³
9λ = λ³
λ² = 9
λ = 3
```

## 例题 7：r.v. X 的分布律为

| X | -1 | 0 | 1 | 2 |
| :--: | :--: | :--: | :--: | :--: |
| P | 0.1 | 0.2 | 0.3 | 0.4 |

`F(x)` 为 X 的分布函数，求 `F(0.5)`

```
F(0.5)
= P{X ≤ 0.5}
= P{x = -1} + P{x = 0}
= 0.1 + 0.2
= 0.3
```

## 例题 8：设离散型 r.v. X 的分布律为

| X | -1 | 0 | 1 | 2 |
| :--: | :--: | :--: | :--: | :--: |
| P | 0.2 | 0.1 | 0.3 | 0.4 |

求 X 的分布函数

```
当 x < -1, P{X < x} = 0
当 -1 ≤ x < 0, P{X < x} = 0.2
当 0 ≤ x < 1, P{X < x} = 0.2 + 0.1 = 0.3
当 1 ≤ x < 2, P{X < x} = 0.2 + 0.1 + 0.3 = 0.6
当 2 ≤ x, P{X < x} = 0.2 + 0.1 + 0.3 + 0.4 = 1

则 F(x) = {
  0, x < -1,
  0.2, -1 ≤ x < 0,
  0.3, 0 ≤ x < 1,
  0.6, 1 ≤ x < 2,
  1, x ≥ 2.
```

## 例题 9：r.v. X 的分布函数为

```
F(x) = {
  0, x < 0,
  x/3, 0 ≤ x < 1,
  x/2, 1 ≤ x < 2,
  1, x ≥ 2.
```

求

1. `P{1/2 < x ≤ 3/2}`

```
公式 2：
= F(3/2) - F(1/2) = 3/4 - 1/6 = 7/12
```

2. `P{x > 1/2}`

```
公式 3：
= 1 - F(1/2) = 1 - 1/6 = 5/6
```

3. `P{x > 3/2}`

```
公式 3：
= 1 - F(3/2) = 1 - 3/4 = 1/4
```

## 例题 14：r.v. X~N(0,1), X 的分布函数为 Φ(x)，求 P(|x| > 2) 的值

```
|x| > a => x > a + x < -a
|x| < a => -a < x < a
```

公式法：

```
P(|x| > 2)
= P{x > 2} + P{x < -2}
= 1 - P{x < 2} + P{x < -2}
// = 1 - Φ((2-0)/1) + Φ((-2-0)/1)
= 1 - Φ(2) + Φ(-2)
= 1 - Φ(2) + 1 - Φ(2)
= 2 - 2Φ(2)
```
