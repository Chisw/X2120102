# 第五章例题

## 例题 1

设 r.v. `X` 的 `E(X),D(X)` 都存在，`E(X) = 10`，`E(X²) = 109`，求 `P{|X -10| ≥ 6} ≤ ?`

```
D(X) = E(X²) - [E(X)]² = 109 - 100 = 9

? = D(X)/ε² = 9/6² = 1/4
```

## 例题 2

投币 1000 次，`X` 表示正面朝上的次数，假设正反面概率相同，求切比雪夫不等式 `P{400 < X < 600} ≥ ？`

```
X~B(1000, 1/2)
E(X) = np = 500
D(X) = np(1 - p) = 250

P{400 < x < 600}
= P{400 - 500 < x -500 < 600 - 500}
= P{-100 < x - 500 < 100}
= P{|x - 500| < 100}

P{|x - 500| < 100} ≥ 1 - D(X)/ε²
? = 1 - 250 /100² = 39/40
```
