## 例题 8

1. `E(X) = 2`, `D(X) = 4`，求 `E(X²)`

```
D(X) = E(X²) - E(X)²
E(X²) = D(X) + E(X)² = 4 + 2² = 8
```

2. `E(X) = -1`，`D(X) = 3`，求 `E(3X² - 2)`

```
D(X) = E(X²) - E(X)²
E(X²) = D(X) - E(X)² = 3 - 1 = 2
E(3X² - 2) = 3E(X²) - 2 = 3 × 2 - 2 = 4
```

## 例题 10

`X,Y` 相互独立，`DX = 4, DY = 7`，求 `D(2X - Y)`

## 例题 11

r.v. `X,Y` `ρₓᵧ = 0.5`，`D(X) = 9`，`D(Y) = 4`，求 `D(3X - Y)`

```
ρₓᵧ = Cov(X,Y) / (√D(X)√D(Y))

Cov(X,Y) = 0.5 × 3 × 2 = 3

D(3X - Y)
= D(3X) + D(Y) - 2Cov(3X,Y)
= 9D(X) + D(Y) - 6Cov(X,Y)
= 81 + 4 - 18
= 67
```

## 例题 12

设二维 r.v. `(X,Y)` 的分布律为

| X\Y | -1 | 0 | 1 |
| :--: | :--: | :--: | :--: |
| -1 | 0 | 0.2 | 0 |
| 0 | 0.2 | 0.2 | 0.2 |
| 1 | 0 | 0.2 | 0 |

1. 求 `ρₓᵧ`

补全边缘分布律

| X\Y | -1 | 0 | 1 | |
| :--: | :--: | :--: | :--: | :--: |
| -1 | 0 | 0.2 | 0 | 0.2 |
| 0 | 0.2 | 0.2 | 0.2 | 0.6 |
| 1 | 0 | 0.2 | 0 | 0.2 |
| | 0.2 | 0.6 | 0.2 |

| X | -1 | 0 | 1 |
| :--: | :--: | :--: | :--: |
| P | 0.2 | 0.6 | 0.2 |

| Y | -1 | 0 | 1 |
| :--: | :--: | :--: | :--: |
| P | 0.2 | 0.6 | 0.2 |

| XY | -1 | 0 | 1 |
| :--: | :--: | :--: | :--: |
| P | 0 | 1 | 0 |

```
E(X) = 0
E(Y) = 0
E(XY) = 0

Cov(X,Y) = E(XY) - E(X)E(Y) = 0

ρₓᵧ = Cov(X,Y) / (√D(X)√D(Y)) = 0
```

2. `X,Y` 是否不相关？是否不独立？

```
ρ = 0，不相关

P{X = -1, Y = -1} ≠ P{X = -1}P{Y = -1}，不独立
```
