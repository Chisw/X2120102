## 例题 9

二维 r.v. `X,Y` 相互独立，且服从参数为 1 的泊松分布，求 `P{X = 1,Y = 2}`

```
= P{X = 1} · P{Y = 2}
= (1¹e⁻¹/1)(1²e⁻¹/2)
= e⁻¹ × e⁻¹ / 2
= e⁻²/2
```

## 例题 10

`X,Y` 相互独立，根据分布律求 `a,b`

| X\Y | 1 | 2 |
| :--: | :--: | :--: |
| 1 | 1/9 | a |
| 2 | 1/6 | 1/3 |
| 3 | 1/18 | b |

```
补全表格边缘
```

| X\Y | 1 | 2 | |
| :--: | :--: | :--: | :--: |
| 1 | 1/9 | a | 1/9 + a |
| 2 | 1/6 | 1/3 | |
| 3 | 1/18 | b | 1/18 + b |
| | 1/3 | | |

```
因为独立，则有 P{X = 1, Y = 1} = P{X = 1}·P{Y = 1}
1/9 = (1/9 + a)(1/3)
a = 1/9

1/18 = (1/18 + b)(1/3)
// 或
b = 1 - 其它之和
```

## 例题 11

r.v. `X,Y` 独立同分布，`X` 的分布律如下，求 `P{X = Y}`

| X | 0 | 1 |
| :--: | :--: | :--: |
| P | 0.4 | 0.6 |

```
同分布：取值、概率相同
```

| Y | 0 | 1 |
| :--: | :--: | :--: |
| P | 0.4 | 0.6 |

```
P{X = Y}
= P{X = 0}·P{Y = 0} + P{X = 1}·P{Y = 1}
= 0.4² + 0.6²
= 0.52
```

## 例题 12

根据一下联合概率分布求 `F(0,1)`

| X\Y | 0 | 1 | 2 |
| :--: | :--: | :--: | :--: |
| -1 | 0.2 | 0.1 | 0.1 |
| 0 | 0 | 0.3 | 0 |
| 2 | 0.1 | 0 | 0.2 |

## 例题 13

二维离散 r.v. 函数分布

| X\Y | 0 | 1 | 2 |
| :--: | :--: | :--: | :--: |
| 0 | 1/4 | 1/6 | 1/8 |
| 1 | 1/4 | 1/8 | 1/12 |

求 `Z = X + Y` 分布律

```
X = 0, 1
Y = 0, 1, 2
Z = 0, 1, 2, 3

P{Z = 0} = P{X + Y = 0} = P{X = 0, Y = 0} = 1/4
P{Z = 1} = P{X + Y = 1} = P{X = 1, Y = 0} + P{X = 0, Y = 1} = 5/12
P{Z = 2} = P{X + Y = 2} = P{X = 0, Y = 2} + P{X = 1, Y = 1} = 1/4
P{Z = 3} = P{X + Y = 3} = P{X = 1, Y = 2} = 1/12
```

得

| Z | 0 | 1 | 2 | 3 |
| :--: | :--: | :--: | :--: | :--: |
| P | 1/4 | 5/12 | 1/4 | 1/12 |
