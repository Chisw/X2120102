### 2.不定积分（反导数、原函数）

函数为 `f(x)` 的不定积分为 `∫f(x)dx`

- 积分符号：`∫`
- 被积函数：`f(x)`
- 被积表达式：`f(x)dx`
- `F'(x) = f(x)`
- `∫f(x)dx = F(x)`

![\int xdx=\frac{1}{2}x^2+C,\int x^2dx=\frac{1}{3}x^3+C](../img/X2120102.04183.00.15.svg)

#### 例题：求 `fx²dx`

```
解：由于 (x³/3)' = x²，所以 x³/3 是 x² 的一个原函数
因此 ∫x²dx = x³/3 + C
```
