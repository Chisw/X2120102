## 例题 5：5 白 3 黑，取两个（不放回），

求相同颜色概率；

```
5/8 × 4/7 + 3/8 × 2/7
```

求第二次为白色概率；

```
5/8 × 4/7 + 3/8 × 5/7
```

## 例题 6：n > 1 张彩票，其中有 1 张奖券，甲乙一次抽取一张，求甲乙中奖概率

```
甲：1/n
乙：1/n × 0 + (n-1)/n × 1/(n-1)
```

## 例题 7：甲乙击中概率分别为 0.9、0.8，求目标被击中概率

```
0.9 × (1 - 0.8) + (1 - 0.9) × 0.8 + 0.9 × 0.8
1 - (1 - 0.9) × (1 - 0.8)
```
