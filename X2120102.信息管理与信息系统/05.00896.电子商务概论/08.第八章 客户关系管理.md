# 第八章 客户关系管理

## 第一节 网上消费者及其行为特征

### 一、消费者心理

营销要以客户为中心，就应该了解客户的需求，处理好与客户的关系

1. 中国网上消费者的特征
  - 性别：均衡
  - 年龄：10 ~ 39 为主体，各方面因素导致年长化
  - 学历：结构稳定
  - 职业：学生占比最高，其次个体、自由职业者，最后企业、公司管理人员
  - 收入：与人均收入提升密不可分
  - 上网时间：人均周上网时间 26.1h，持续增长
  - 对互联网信任程度：大幅提高
  - 网络分享：积极
  - 对互联网依赖程度：学历越高，占比越大

2. 网上消费者心理特征
  - 消费者主体意识加强：消费时尚、个性化
  - 消费者理性意识加强
  - 快捷，方便的心理加强
  - 消费者对价格便宜的需求加强

### 二、影响网上消费者需求的因素

1. 收入：包括人均国民收人
2. 消费者偏好
3. 其他物品价格：替代品、互补品、非电子商务产品

### 三、网上消费者的购买行为

1. 分类
  - 特定购买：购买前目标明确，了解行情
  - 计划购买：不了解行情
  - 提醒购买：通过广告等进入

2. 决策过程
  - 认知问题：明确购买对象
  - 搜寻信息：多角度比较
  - 信息评价与决策：筛选信息
  - 购买行为
  - 购买后行为：评价

3. 决策的基本原则
  - 最大满意原则：理想化的
  - 相对满意原则
  - 遗憾最小原则
  - 预期满意原则：加快决策进程

## 第二节 客户关系管理理论

### 一、客户关系管理的概念

CRM 企业为赢取新客户、维持老客户，通过不断地沟通和了解客户，以不断增进企业利润为目的，达到影响客户购买能力的管理办法

### 二、电子商务下客户关系管理的特点

1. 采用网络信息技术
2. 技术系统和商业流程的融合
3. 将客户信息转变为客户关系：单向变双向

### 三、客户关系管理的网络营销活动

四个阶段：

1. 客户选择：确定公司销售的对象
2. 客户获得：新客、旧客在线买
3. 客户保留：保留客户、使其经常访问
4. 客户延伸：增加范围或种类

## 第三节 客户关系管理的新应用

### 一、客户细分和定位

5 个要素：

1. 识别网上客户生命周期的不同阶段
  - 首次访问，cookies
  - 再次访问
  - 可能客户：注册
  - 一般客户：购买
  - 长久客户：持续购买

2. 识别客户特征
  - 年龄、性别、地区、收人、对互联网的依赖和信任程度
  - 括企业的规模和企业所处的行业或领域

3. 识别客户行为和购买偏好
  - 访问页面
  - 停留时间
  - 浏览产品页面
  - 收藏页面
  - 访问间隔时间
  - 最近一次的购买详情
  - 所有的购买次数
  - 购买总额
  - 购买产品的类型

4. 识别客户的渠道偏好
  - PC
  - 移动端

5. 识别客户的信息形式偏好
  - 理性展示：邮件
  - 图像、非正式

### 二、在线个性化推荐

通过信息过滤和数据挖掘等技术将商品、服务或者链接等推荐给可能需要的用户

1. 作用
  - 有效地改善产品的市场表现、提升销售绩效
  - 促进基于消费者参与的产品创新、培养顾客忠诚、形成差异化竞争优势
  - 将网站的浏览者转变为购买者
  - 提高网站销售能力

2. 形式
  - 再销售：相似的产品
  - 交叉销售：密切相关但非必须的附加产品
  - 向上销售：更昂贵的产品
  - 推荐：客户朋友圈

### 三、网上社区

是网络营销的主要营销手段之一

把具有共同兴趣的访问者集中到一个虚拟空同，让网民产生互动、情感维系区资讯分享，达到成员之间相互沟通的目的，从而实现商品的营销效果

综合性的社区、专业性的社区

## 第四节 客户关系管理技术

### 一、CRM 系统

是一套先进的管理模式，其实施要取得成功，必须有强大的技术和工具支持

基于网络、通讯、计算机等信息技术

实现不同部门无缝连接，协助完成客户管理任务

客户联系强度：链接频率、链接深度、链接范围

1. 要素
  - 面对客户、接触客户：
    - 尽可能多的接入形式
    - 洞察客户需求
    - 注重潜力客户
    - 提供个性服务界面
    - 提供客户放心的安全环境
  - 处理客户问题：
    - 尽可能高效的解决
    - 寻找一套适用性较好的客户关系管理系统方案
    - 将客户关系管理融入企业整个价值链

2. 主要功能
  - 客户信息管理 CIM：档案
  - 销售过程管理 SFA：行为管理
  - 营销自动化 MA：管理广告、电子邮件
  - 客户服务与支持管理 CSS
  - 客户分析 CA：数据挖掘

3. 实施
  - 管理方面：基础
  - 技术支持：效率
  - 流程：
    - 确立业务计划
    - 建立 CRM 团队
    - 评估销售、服务过程
    - 明确实际需求
    - 选择供应商
    - 开发与部署

### 二、数据挖掘技术

从海量的信息中得到真正有价值的信息，以指导他们的决策

1. 有关概念
  - 数据仓库：面向主题、集成性、稳定性和时变性
  - 数据挖掘：选择、处理、挖掘、分析
  - 数据挖掘模型：
    - 分类：实例分配到组，寻找共性
    - 回归：利用现有数据预测其他值
    - 时间序列：也是预测，加入时间值
    - 分组：跟踪发现区别或类似的组
    - 联合分析：揭示一起发生的事情
    - 序列发现：衡量关联性并预测日后扩展销售

2. 客户关系管理系统中的数据流动
  - 通过市场调查、会员注册信息等方式收集的信息存储于数据仓库
  - 系统将经数据挖掘后提取的数据与知识库同其他数据存储中的数据提供给 CRM 应用系统

```
数据收集：市场调查、注册信息
 ↓
数据仓库：挖掘、存储
 ↓
CRM 应用
```

3. Web 数据挖掘
  - 来源：
    - 服务器数据
    - 查询数据
    - 在线市场数据
    - Web 页面
    - 超链接
    - 客户登记信息
  - 类型：
    - Web 内容挖掘：文本、图像、音视频
    - Web 结构挖掘：超链接关系、页面结构
    - Web 使用模式挖掘：日志挖掘

4. 数据挖掘在 CRM 中的作用
  - 发现潜在客户
  - 延长客户的驻留

### 三、大数据与大数据营销

1. 大数据的概念
  - 数据爆发增长
  - 多样、价值密度低、实时性
  - 需要新处理模式才能具有更强的决策力、洞察发现力和流程优化能力的海量、高增长率和多样化的信息资产

2. 大数据营销
  - 通过互联网采集大量的行为数据，在大量云计算和大数据分析的基础上，描绘、预测、分析、指引消费者行为，从而帮助企业制定有针对性商业策略的营销过程
  - 特点：
    - 多平台化数据采集
    - 强调时效性：及时
    - 个性化营销：定制
    - 性价比高：准确
    - 关联性强：深度互动
