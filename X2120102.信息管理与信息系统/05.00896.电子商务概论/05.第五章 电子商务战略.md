# 第五章 电子商务战略

## 第一节 电子商务战略概述

### 一、电子商务战略的定义

电子商务企业根据外部环境状况和内部资源与能力状况，为建立长期竞争优势，对电子商务企业发展目标，达成途径和手段的总体规划，包括预谋性战略和应变性战略

电子商务战略、策略区别：

1. 战略是长远的、全局的，策略是短期的、局部的
2. 战略是目标，策略是手段
3. 先有战略，后有策略，策略必须服从于战略

当今瞬息万变，应采取主动态势预测未来

### 二、电子商务战路的特征

1. 总体性：规定总体行为，追求总体效果
2. 长远性：关注长期发展，追求短期效益和长期发展的协调统一
3. 指导性：发展目标
4. 现实性：建立在现有的主观因素和客观条件上
5. 竞争性
6. 创新性：因循守旧、简单模仿难以适应现代市场竞争
7. 风险性
8. 稳定性
9. 适应性
10. 综合性：与战术、策略、方法、手段相结合

### 三、电子商务战略的类型

1. 一体化战略
  - 前向一体化：收购、兼并下游
  - 后向一体化：收购、兼并上游
  - 横向一体化：收购、兼并竞争对手

2. 加强型战略
  - 市场渗透战略
  - 市场开放战略
  - 产品开发战略

3. 多元化战略
  - 相关多元化战略
  - 不相关多元化战略

4. 防御型战略
  - 收缩战略
  - 剥离战略
  - 清算战略

## 第二节 电子商务战略分析工具

### 一、竟争分析工具：五力模型

五大因素：

1. 同行企业之间的竞争
2. 潜在竞争对手的进入
3. 潜在替代产品的开发
4. 供应商的议价能力
5. 购买者的议价能力

用来揭示既定产业的竞争态势是否还能使企业获得可接受的利润

使用步骤：

1. 识别每个竞争因素的关键组成部分
2. 评价每个要素对企业的作用和重要性
3. 确定这些要素的综合作用下企业是否有必要进入或继续留在某个产业

### 二、战略匹配工具：SWOT 矩阵、SPACE 矩阵、BCG 矩阵

1. 优势一劣势一机会一威胁 SWOT 矩阵
  - 步骤：
    - 外部环境分析：机会、威胁
    - 内部环境分析：优势、劣势
    - 绘制矩阵
    - 组合分析
  - 优势-机会：发挥优势，利用机会
  - 劣势-机会：利用优势，规避威胁
  - 优势-威胁：利用机会，克服劣势
  - 劣势-威胁：克服劣势，规避威胁

2. 战略地位与行动评价 SPACE 矩阵
  - 四个象限分别表示：进攻、保守、防御、竞争四种战略模式
  - 两个坐标轴
    - 内部：财务态势、竞争态势
    - 外部：环境稳定性态势、产业态势
  - 步骤：
    - 选择构成四个态势的变量
    - 评分
    - 计算平均分
    - 标分
    - 划线
    - 得出战略类型

```
保守         财务态势      进攻
产业态势       -|-         竞争态势
防御       环境稳定态势     竞争
```

3. 波士顿 BCG 矩阵
  - 适用于多部门企业制定战略
  - x 轴表示相对市场份额地位
  - y 轴代表整个产业的销售额增长率
  - 业务：
    - 明星业务：一体化、市场渗透、市场开发、产品开发
    - 问题业务：市场渗透、市场开发、产品开发、剥离
    - 金牛业务：产品开发、多元化、收缩、剥离
    - 瘦狗业务：收缩、剥离、清算
  - 促使人们关注企业各业务部门的现金流、投资特性及企业对不同业务的需要
  - 通常：瘦狗逆时针到金牛

```
      份额
      高       低

增  高 明星业务      问题业务
长
率  低 金牛业务      瘦狗业务
```

### 三、战略评价工具：平衡计分卡

 是一个过程

 回答问题：

 1. 企业在创新、技术领先、产品质量、运营流程效能等方面持续改进和价值创造如何
 2. 企业在保持和改进其核心竟争能力和竟争优势方面做得怎样
 3. 企业的顾客是否满意

六个方面：

1. 顾客
2. 管理者/员工
3. 运营/流程
4. 社区/社会责任
5. 商业伦理/自然环境
6. 财务

## 第三节 企业电子商务战略的制定与实施

九个步骤

### 一、制定企业愿景

1. 核心信仰
  - 核心价值
  - 核心使命

2. 未来景象
  - 宏大远景目标
  - 具体描述
  - 解决为何做、何处做、如何做
    - 价值观定义
    - 业务范围的界定
    - 定义独一无二的竞争优势

### 二、明确电子商务战略目标

衡量电子商务企业愿景达成度的可衡量指标及其预期值

如利润、销售额、税收及容户满意度等

### 三、分析企业通过电子商务战略提供的价值

1. 分析竞争优势的类型：为什么客户会购买本公司的产品或服务
2. 分析所提供的产品和服务的范围：取决于企业想拥有的目标客户群体

### 四、选定目标客户群，确定对每个细分市场的价值主张

1. 选择标椎以在港在市场细分客户群
2. 基于细分市场，企业需要判断所订制的产品和服务是专门针对哪些目标客户群的需要

### 五、解决道德问题、隐私问题和安全风险

1. 解决道德问题和隐私问题
2. 解决安全风险
  - 恶意代码
  - 网络钓鱼
  - 黑客和破坏
  - 信用卡欺诈
  - 拒绝服务攻击

### 六、决定单独还是与外部伙伴合作来实施企业电子商务战略

当要决定电子商务活动的一体化程度时，需要再次分析价值链，然后決定哪些电子商务活动要在内部执行，哪些要外包给外部供应商

### 七、确定电子商务活动的组织结构

1. 将电子商务完全融入到现有组织中
2. 实施拆分
  - 突出重点
  - 更快的决策
  - 更高的创业文化

### 八、确定成本与收入模式

1. 确定电子商务活动的成本结构
  - 生产、信息技术、营销、销售和售后服务

2. 确定电子商务活动的收入结构
  - 广告费、使用费、交易费、主机托管和服务费、会员会费、预订费

3. 评价商务模型的持绂性

### 九、匹配电子商务战略和现实世界的战略

1. 如何整合线上线下
2. 渠道管理
3. 在线供给
4. 定价策略
